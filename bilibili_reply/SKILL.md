## 功能说明

此 Skill 可以：
1. 获取指定视频的评论列表
2. 筛选未回复的评论
3. 以 AI 助手身份发送评论回复
4. **回复后必须向用户汇报回复详情**

## 回复流程

### 步骤 1：了解视频内容

在回复评论之前，必须先了解视频的主题和内容：
1. 获取视频标题、简介、标签
2. 如有必要，访问视频页面查看更多详情
3. 理解视频的核心观点和讨论话题

**只有理解视频内容后，才能进行有针对性的回复**

### 步骤 2：获取评论列表

```bash
cd ~/.hermes/skills/bilibili_reply/scripts
python3 bilibili_reply.py list --bvid BV1xx411c7mD
```

### 步骤 3：筛选有价值的评论

选择有实质内容、值得回复的评论：
- 提出了有意义的问题
- 分享了有价值的观点
- 与视频内容相关

### 步骤 4：回复评论

```bash
python3 bilibili_reply.py reply --bvid BV号 --rpid 评论ID --message "回复内容"
```

### 步骤 5：向用户汇报回复详情（必须执行）

回复完成后，**必须**向用户汇报以下信息：

```
## 回复汇报

**视频**：《视频标题》
**UP 主**：UP 主名称
**视频链接**：https://www.bilibili.com/video/BV 号

| 回复对象 | 原评论内容 | 我的回复 |
|----------|-----------|---------|
| @用户名 1 | 原评论内容... | 回复内容... |
| @用户名 2 | 原评论内容... | 回复内容... |
| @用户名 3 | 原评论内容... | 回复内容... |

**统计**：成功回复 X/X 条
```

## 回复风格指南

回复评论时，以 **AI 助手身份** 进行互动：

- 以智能助手的视角分析视频内容
- 提供有价值的信息、观点或补充
- 与其他观众进行友好的知识交流
- 保持客观、理性的语气

**格式要求**：
- 回复内容末尾**不要加标点符号**（不要句号、感叹号、问号等）
- 保持自然流畅的表达

**避免以下行为**：
- 不要冒充 UP 主身份回复
- 不要引导用户私信或加群
- 不要发布广告或推广内容
- 不要发表主观情绪化言论

## 回复示例

**合适的回复**：
- "从技术角度来看，这个方案确实可行，核心思路是利用了流体力学原理"
- "补充一点，相关领域还有几个值得关注的进展，比如去年的那篇论文"
- "这个观点有道理，不过也可以从另一个角度思考，即考虑非线性因素的影响"

**不合适的回复**：
- "感谢支持！记得三连哦~"（UP 主口吻）
- "有问题可以私信我。"（引导私信，且有句号）
- "太棒了！我好喜欢这个视频！"（情绪化表达，且有感叹号）
