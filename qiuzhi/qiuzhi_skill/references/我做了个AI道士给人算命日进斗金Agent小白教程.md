# 我做了个AI道士给人算命...日进斗金【Agent小白教程】

## 核心观点

用阿里云百炼平台，通过无代码方式搭建一个能赚钱的Agent。只需鼠标点点点就能完成：提示词、知识库、MCP、插件的配置，还能接入支付宝收款接口。

## 做赚钱Agent的四步

1. 找到一个好的市场需求
2. 找个无代码平台搭建Agent
3. 给他接上一个收款接口
4. 发布出去让他赚钱

## 案例：赛博道士Agent

### 第一步：新建Agent
1. 进入阿里云百炼
2. 点击应用 → 应用管理 → 新增应用
3. 创建Agent，选择大模型
4. 写提示词（基本设定）
5. 点击"优化"自动扩展成工程化提示词
6. 手动调整完善

### 第二步：配置知识库
**上传的资料**：
- 周易八卦卦象解析
- 梅花易数
- 风水面相
- 星座运势
- 周公解梦

**操作流程**：
1. 应用数据 → 导入数据 → 拖入资料
2. 知识库 → 创建知识库 → 勾选数据
3. 回到Agent → 打开知识库 → 添加知识库
4. 修改提示词要求根据知识库回答

### 第三步：配置工具（MCP）
**MCP广场可开通的工具**：
- 支付宝（收款）
- 作图表
- 数字人
- 证券之星
- 生图
- 地图
- 全网部署
- 等等...

**操作流程**：
1. MCP广场 → 点击开通
2. 申请密钥粘贴进来
3. 一键添加智能体

**其他能力**：
- 视觉能力（接收图片）
- 接收文件
- 联网搜索
- 调用其他Agent

### 第四步：发布
1. 点击发布
2. 勾选打赏功能
3. 申请上架到Agent广场

**发布渠道**：
- 公众号
- 默认Web UI
- 多模态套件（接入硬件）

## 赛博道士Agent功能演示

- 根据生肖说出今年运势
- 算姻缘要收费20元
- 调用支付宝接口验证付款
- 付款后配对姻缘、画长相
- 送图表分析命格
- 部署敲木鱼功德网站积德

## 阿里云百炼平台特点

- **无代码开发**：提示词、知识库、MCP、插件都用点点点完成
- **现成支付宝MCP**：直接接到Agent里收款
- **Agent模板广场**：各行各业模板复制即用
- **打赏功能**：开通后可收打赏

## 核心价值

发挥创造力，找到看好的需求，在百炼上做成能收钱的Agent。
