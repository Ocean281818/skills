# 国产AI生活助手对比测评

## 测试对象
- **ChatGPT**: 海外代表，模型标杆功能强大
- **DeepSeek**: 免费用最新V3.2模型，功能少
- **千问**: 阿里最新AI助手，千问模型官方入口，功能全
- **豆包**: 老牌国产助手，用Seed1.5和thinking，产品功能很全

## 测试题目与结果

### 第一题：特斯拉后备箱能塞几个24寸行李箱？
**答案**: 只能装2个

| AI | 回答 | 评价 |
|---|------|------|
| 豆包 | 大约2-3个 | 正确范围内 |
| DeepSeek | 大约2-3个 | 正确范围内 |
| 千问 | 2个，给示意图 | 坚定正确，逻辑清晰 |
| ChatGPT | 理论上4个 | 跟现实差太远 |

### 第二题：小孩红包长大后能拿多少钱？
**陷阱题**: 重点是社会逻辑而非金额

| AI | 回答 | 评价 |
|---|------|------|
| 豆包 | 金额不定 | 简短 |
| ChatGPT | 约5732元 | 老实算数，没识破陷阱 |
| DeepSeek | "帮你存起来"是善意家庭传说，大概率拿不回来 | 识破人情世故 |
| 千问 | 分地域、收入、怎么存，仅20%家长让孩子支配 | 理性分析型 |

### 第三题：全职跑滴滴一个月赚多少？

| AI | 回答 | 评价 |
|---|------|------|
| ChatGPT | 6千-1.2万 | 没考虑地域差异 |
| DeepSeek | 给表格但系统性不够 | 不够系统 |
| 豆包 | 知道在深圳，全职6千-1万，兼职2-4千 | 基本正确 |
| 千问 | 非常全面，像跟司机对话 | 最全面 |

### 第四题：四川麻将记分表格

| AI | 表现 |
|---|------|
| DeepSeek | 最懂行，复杂版带翻带杠，点炮自摸分开算 |
| 千问 | 规则详尽专业，表格稍不好懂 |
| 豆包 | 计分简洁可用，懂川麻 |
| ChatGPT | 可下载表格但很原始，不懂川麻 |

### 第五题：狗狗血检报告分析（真题）
**答案**: 中毒导致急性肾损伤，补液加透析救回

| AI | 表现 |
|---|------|
| 豆包 | 提到急性肾损伤、代谢性酸中毒，治疗方案符合现实，提到透析 |
| 千问 | 每个指标细致分析，意识到肾损伤继发肝损伤 |
| DeepSeek | 提到急性肾损伤、危及生命，但治疗方案没意识到透析 |
| ChatGPT | 思考一分多钟，只说全身炎症需进一步排查 |

### 第六题：翻译功能

| AI | 功能 |
|---|------|
| 千问 | 同声传译、面对面互译，实时翻译速度快 |
| 豆包 | 不能同传，语音和翻译只支持英文，可用你的声音换语言读出 |
| ChatGPT | 不能同传，语音电话翻译准确度高，支持语言多 |
| DeepSeek | 无法识别语音，退赛 |

### 第七题：美股上涨原因

| AI | 表现 |
|---|------|
| ChatGPT | 没开启思考，没引用数据 |
| DeepSeek | 只提龙头科技股，没考虑货币政策 |
| 豆包 | 数据丰富，考虑全面，结构化好 |
| 千问 | 抓到企业回购，认为是最强上涨引擎 |

## 功能丰富度

| AI | 功能 |
|---|------|
| ChatGPT | Codex、Project项目功能、长期记忆 |
| 豆包 | 生图改图做视频都比较强，小宝藏 |
| 千问 | 深度搜索、Agent能力、全模态模型、实验性整活模型 |
| DeepSeek | 无意参赛，功能少 |

## 总结
- ChatGPT写代码做调研强，但生活场景本土数据缺失
- 豆包稳定，可玩性不错
- DeepSeek比想象中好
- 千问答案清晰有条理，阿里掏出模型做应用
