# 手把手彻底学会 Agent Skills【小白教程】

## 核心观点

**Skill**是Agent的技能包，类似于人类的技能——包含流程、配方、工具和材料。Claude的Skill采用**按需加载的三层结构**设计，能大幅节省token并提高AI回答精准度。

## Skill的本质

### 与人类技能类比
- 厨师有炒菜技能、处理食材技能、摆盘技能
- 每个技能包含：流程（先炒什么后放什么）、配方（油温多高、盐放多少）、工具（煤气灶）、独家材料（秘制辣椒酱）

### Agent Skill的组成
- **skill.md** - 流程和配方
- **references** - 参考资料
- **scripts** - 可执行脚本（工具）
- **assets** - 资源文件（材料）

## Skill的核心设计：按需加载三层结构

### 第一层：元信息（始终加载）
- 每次对话AI都会看所有skill的元信息
- 包含：skill名字、描述
- 作用：像一个目录，让AI知道自己有哪些技能

### 第二层：指令层（按需加载）
- 只有当AI决定要用这个skill时才加载
- 完整的skill.md文件内容
- 包含具体的执行指令

### 第三层：资源层（进一步按需加载）
- references里的参考资料
- scripts里的脚本
- assets里的资源
- 只有任务需要时才加载

### 关键优势
- **节省token**：不需要的信息不加载
- **更精准**：没有其他提示词干扰
- **脚本只执行不读取**：代码不占用token

## Skill文件结构

```
.claude/skills/
└── 你的skill名字/
    ├── skill.md          # 必需：元信息+指令
    ├── references/       # 可选：参考资料
    │   ├── 实体物料规格.md
    │   └── 社交媒体规格.md
    ├── scripts/          # 可选：可执行脚本
    │   └── 生图脚本.py
    └── assets/           # 可选：资源文件
        └── logo.png
```

## skill.md文件格式

```markdown
---
name: 秋知创意
description: 做创意物料用的，当用户说要做海报等物料时触发
---

# 指令部分
我们的餐厅叫做秋知餐厅
品牌风格要求：...
输出格式：...
如果要做线下物料，请读取 references/实体物料规格.md
如果要求直接生成图片，请运行 scripts/生图脚本.py
参考图在 assets/ 文件夹
```

## 创建Skill的方法

### 方法一：手动创建
1. 在`.claude/skills/`下创建文件夹
2. 创建skill.md文件，写入元信息和指令
3. 按需添加references、scripts、assets

### 方法二：用Skill创建器（推荐）
- 使用秋知Skill Creator
- 只需回答AI的选择题
- AI自动生成所有skill文件

## 实用Skill案例

### 案例：秋知创意Skill
- 输入：帮我做一个促销海报
- 输出：符合品牌风格、带logo的物料图片
- 功能：自动调用Nano Banana API生图

### 常用必备Skill
- 做PPT
- 处理文档/Excel/PDF
- 文章配图
- 前端设计（大幅提升网页效果）
- 动画生成

## 创建自己Skill的建议

> 一个游戏英雄也只需要四个技能QWER就能杀遍全场，最有效的还是把你最高频做的几件事打磨成独家稳定的skill。

适合做成Skill的场景：
- **周报**：AI采访你后出周报
- **备课**：给出课题，自动生成课件、习题、PPT
- **配图**：给文章自动按风格配图
- **审合同**：按规矩自动批阅合同写备注

## 核心价值

大多数人不需要成为技能开发者，只要把自己掌握的小技能交给AI，让它替你重复劳动。
