# 爆肝10亿token，我给OpenClaw做了个龙虾管家

## 核心观点

作者爆肝半个月、消耗10亿token，开发了**QClaw（龙虾管家）**——一个OpenClaw的可视化管理工具，让小白用户也能一键安装和管理OpenClaw（龙虾）。

## OpenClaw六大核心能力

1. **啥都能干** - 权限比任意AI产品都大，能操作电脑、命令行、浏览器、桌面应用
2. **永久记忆** - 记忆以文件总结形式保存在本地，越来越懂你
3. **自我进化** - 犯过的错主动写进文件，下次就会包含在上下文里
4. **7×24小时主动干活** - 每30分钟自动查看心跳文档，可设置定时任务
5. **技能生态庞大** - 一句话就能外接新能力
6. **多平台连接** - 通过网关连接飞书、钉钉、QQ等通讯软件

## 设备与模型选择建议

### 设备推荐排序
1. Mac本地部署（最佳体验）
2. Windows本地部署
3. 云服务器
4. 安卓手机/树莓派（可行但体验较差）

### 模型搭配建议
- **主模型**：GPT会员用Codex免费额度，或打折渠道的Claude
- **干活模型**：Coding Plan套餐当牛马
- **省钱Skill**：根据任务难度自动切换模型

## QClaw一键安装流程

1. 下载QClaw安装包（官网）
2. 双击安装，自动检测环境
3. 选择模型服务商，填写API Key
4. 选择连接的通讯软件（飞书/钉钉/QQ）
5. 配置飞书应用（App ID和App Secret）
6. 选择群聊策略
7. 启用内置Skill
8. 配置Hook选项
9. 完成安装，开始对话

## QClaw管理面板功能

### 小窗模式（高频使用）
- 查看龙虾存活状态
- 今日token消耗统计
- 重启网关/自动修复/回滚配置
- 停止任务（防止浪费token）
- 切换模型/添加新模型
- 设置每日预算限额

### 展开模式（细致管理）
- 管理每个会话
- Skill管理
- 记忆管理

## 五大实用玩法

1. **定时任务** - 自动检查更新、发早报晚报、安排行程
2. **信息采集** - 实时抓取视频评论、网页内容
3. **后台开发** - 用Codex开发产品、自动提交GitHub
4. **流程自动化** - 报销审核、立项、采购等企业流程
5. **联动家居** - 控制智能设备、音箱等

## 安全提醒

- 不建议将日常使用的龙虾放到公开群聊
- 提示词注入风险如猫鼠游戏，可能泄露秘密
- 需设置护栏和约束
