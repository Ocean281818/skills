# 躺着办公这么爽？AI时代好东西分享1.0

## 核心观点

AI时代的工作方式正在改变：从执行者变成管理者。AI包揽具体执行，人类更多做判断和决策。**身体松弛、精神专注**的状态更适合需要创造力的未来办公方式。

## AI时代必备四件装备

### 1. 多屏幕
- **原因**：Agent完成任务需要时间，不可能干等着串行汇报
- **用法**：同时开好几个并行任务
  - 龙虾：汇总杂乱信息成文档
  - Claude Code：拉框架、想主题
  - OpenAI Atlas浏览器：探索工具官网、总结汇报
- **状态**：像监工一样，这个好了检查这个，那个好了检查那个
- **建议**：至少十几个窗口的显示器

### 2. 放松款人体工学椅
- **核心理念**：后仰办公，脊柱压力更放松
- **特点**：
  - 千人千背，颈枕、腰背、坐垫三方贴合
  - 动态支撑：跟随动作变化
  - 后仰档位可调
  - 海绵材质，舒适不硌
  - 坐垫通风：4000转/分钟风扇抽气散热
  - 脊柱拉伸按摩功能
- **推荐**：清闲Pro（适合AI办公的"躺平"状态）

### 3. 麦克风
- **原因**：AI语音输入法产品成熟，基本免费
- **优势**：
  - 效率比打字高5倍
  - 自动理解成通顺文字，带排版
  - 口述给AI交代任务比敲提示词轻松
- **团队理念**：
  - 自己输出用语音（说话比打字效率高）
  - 给别人看/看别人的用文字（看文字比听说话效率高）
  - 效率double

### 4. 家庭服务器
- **本质**：长期运行的计算机
- **用途**：
  - **Agent运行**：龙虾24小时运行，数据保存，工具积攒
  - **算力保障**：本地部署大模型，应对API挂/欠费/涨价/隐私需求
  - **生活中枢**：监控、智能设备、机器人等都需要算力
- **未来**：24小时在线，为生产力系统和生活系统提供服务

## AI时代办公图景

> 服务器里面跑着Agent，自己半躺在椅子上，通过麦克风下达指令，看着各个屏幕上的AI执行结果。每个人都会更像一个管理者。

## 核心洞察

> AI能够完整干活的那一天会比你想的来得快，但人类去适应这种跳出执行、去做判断的管理思维，可能比想象中要慢。

**先躺为敬**，希望大家都能在AI时代舒适高效轻松地办公。
