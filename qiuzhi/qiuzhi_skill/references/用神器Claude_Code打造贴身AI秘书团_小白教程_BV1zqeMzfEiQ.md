# Claude Code打造贴身AI秘书团教程

## Claude Code是什么
- Anthropic今年5月发布的编程代理
- 编程领域Agent大哥大，Anthropic 70%代码用它写
- 最被低估用法：做思维搭子

## 三种AI对比
| 类型 | 特点 |
|------|------|
| 网页AI | 问询处机器人，没手脚，不认识你 |
| 打包Agent | 前台小姐姐机器人，能打电话敲电脑，但不认识你 |
| Claude Code | 贴身秘书机器人，了解你，工具箱无限，能带团队 |

## 三步打造秘书团

### 第一步：安装配置
1. 打开Cursor（或Trae、Kiro），开空文件夹
2. 聊天框选Agent模式
3. 发Claude Code官方文档，说"帮我安装Claude Code"
4. 换大脑：用国产GLM4.5（或其他模型）
   - 获取API key
   - 复制命令替换key

### 第二步：设计团队
创建构想MD文件，设计秘书分工：
- **新闻秋**: 每天早上搜AI新闻，总结写文档，发飞书
- **穿搭秋**: 问场合、查天气、给穿搭建议
- **教练秋**: 记录体重健康数据，推荐运动和饮食
- **日报秋**: 问安排，规划日程早报，发工作群
- **反思秋**: 晚上引导反思复盘

让Claude Code完善想法：
- 圈文档问"我想设置几个sub agents，帮我完善"
- 参考生成的设定和工具需求
- 填写个人信息模板

### 第三步：配置工具（MCP）
1. 去工具超市Smithery挑工具
2. 例：穿搭球需要即梦生图
   - 搜即梦，选Cloud Code
   - 获取session ID（登录后右键检查→application→cookies）
   - 复制命令粘贴执行
3. 飞书等复杂MCP：问官方文档AI或豆包侧边栏
4. 检查：`claude mcp list` 确认connected

## 创建Sub Agent
1. 输入`claude`
2. `/agent` → create → 选择项目
3. 回答希望agent干什么
4. 配工具（默认全用）
5. 选模型、挑颜色
6. 重复创建不同秘书

## 初始化
`/init` 生成CLAUDE.md规章制度
- 概述项目结构
- 秘书团成员
- 文件夹用途
- 遵循原则

## 实战演示

### 早间新闻
- 打招呼"雷猴啊早上好"
- 自动召唤新闻秋搜新闻
- 飞书收到新闻早报
- 每日记录文件夹出详细简报

### 穿搭建议
- 告知场合（白天开会，晚上晚宴）
- 生成穿搭建议文档
- 按形象出参考照片

### 日报规划
- 敲入今天安排
- 细化注意事项、关键决策点
- 给座右铭

### 健康管理
- 记录体重、身体状态
- 生成消肿方案、饮食建议
- 教练寄语

## 总结
- 过去花时间找备忘录、体重记录app、日报模板
- 现在可以自己定制听命于你、有智力的工具
- 掌握Claude Code sub agent非编程玩法
