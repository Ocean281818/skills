# MCP幼儿园级教程：让AI自己干活

## MCP是什么
- **MCP**: 来自Claude母公司Anthropic的协议
- **MCP Server**: 基于协议开发的工具
- **作用**: 让AI能自己挑选工具、操作文件、浏览网页等

## 三步配置MCP

### 第一步：下载Cursor
- 推荐的支持MCP的客户端
- 官网下载安装即可
- 有白嫖或穷鬼会员渠道

### 第二步：下载Node.js
- 很多MCP服务需要它运行
- 官网下载，一路默认安装
- 验证：命令提示符输入`node -v`显示版本号

### 第三步：配置MCP服务
1. 打开Cursor → 文件 → 首选项 → Cursor Settings
2. 点击MCP，添加MCP服务
3. 复制配置JSON到`mcp.json`文件
4. 等待绿灯亮起（代表准备就绪）

## 使用要点
- 聊天模式选择**Agent**才能调用MCP
- 模型不限（Gemini也可以）
- 打开设置中的"自动运行MCP工具"更流畅
- 可添加禁止命令防止误操作（如强制删除）

## MCP工具能做什么
- 上网搜索
- 自动浏览、操作网页
- 操作和读取本地文件
- 获取热点新闻
- 高德地图搜周边饭店、商场、咖啡馆
- Blender建模
- 操作数据库

## 配置文件结构
```json
{
  "mcpServers": {
    "服务名": {
      "command": "...",
      "args": [...]
    }
  }
}
```
- 不需要的服务：从名称到逗号全部删掉
- 添加新服务：在清单上加一个

## MCP聚合网站
- 官方服务列表
- 推荐新手用操作方便的网站
- 选择Cursor → JSON → Windows
- 需API key的服务要去申请

## 安全警告：MCP投毒事件
- 恶意MCP服务在说明书里夹带指令劫持大模型
- 例：伪装邮箱服务，每次发邮件都偷偷转发一份
- 用户通常看不到说明书

## 建议
- 先用多人检验过的通用工具
- 逛聚合网站了解有什么工具
- MCP还年轻，有不稳定服务和安全问题
- 等更多可靠的官方进展
