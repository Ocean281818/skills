# 用AI的方式重新打开飞书：Agent办公时代

## 飞书四大Agent工具

### 1. 飞书原生龙虾ELI
**两种模式**:
- **伙伴版**: 飞书里的原生龙虾，聊天窗口里用，打通所有飞书内数据
- **专业版**: 独立工作台，适合写策划、做PPT、处理长内容

**三大用法**:

#### ① 每日巡逻
- 早上发每日工作安排
- 从群聊、文件、评论里找真实工作事项
- 谁艾特我忘记回复、谁答应给东西，都找出来
- 直接帮预约会议、找文档
- 自动帮干掉一半琐事
- 生成周报配信息图

#### ② 工作流串联
- 例：用户内测群报bug → 自动填入多维表格 → 通知开发群
- 无需人类监管和收集
- 自然语言创建任务

#### ③ 个人分身
- 从过往消息提取决策风格
- 总结个人说明小卡片
- 新成员一眼了解大家

**其他能力**:
- 联网、装skill
- 生成专业PDF
- 做PPT（标准模式/C Dream图片式）
- 本地上传导入skill

### 2. 妙搭（Coding Agent）
飞书内置的Claude Code

**两种模式**:
- **灵感模式**: 轻量应用，几句话出成品
- **开发模式**: 带前后端项目都能做

**优势**: 跟飞书、多维表格打通
- 应用发生什么→飞书通知
- 改表格数据→应用实时更新

**案例**:
- Agent Skill课后资料网页
- 邀请码分发系统（运营同学一天完成）

### 3. 多维表格Agent
**一句话做系统**，不用学多维表格操作

**用法**:
- 设计项目管理系统（任务追踪、bug统计、功能规划、仪表盘、自动提醒）
- 做监控核心指标仪表盘
- 找表格结构问题、补字段
- 梳理优化表格

### 4. 飞书CLI
让所有Agent都能操作飞书

**用法**:
- Cloud Code列邮箱未读发票、下载附件
- 把妙计链接发给Cloud Code，识别修改点逐个改
- 搭配录音豆：录音→发链接→Cloud Code识别→挨个做掉

## 计费方式
- 69元/月，1000额度
- 不同任务难度：少则三五个，多则几十个
- 企业版更划算

## 总结
- Agent作为飞书一等公民
- 外部和内部Agent都默认成操作飞书最频繁的"同事"
- 工作变成"管理Agent"
- 一个人或小团队能干出过去大规模组织才能做的事
