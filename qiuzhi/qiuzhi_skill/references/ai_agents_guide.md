# AI Agent 技术指南

## OpenClaw (龙虾) 完全指南

### 什么是OpenClaw
OpenClaw是一个强大的AI Agent框架，具有以下核心能力：
- **权限大**：可以操作电脑、命令行、浏览器、桌面应用
- **永久记忆**：以文件形式保存记忆，越用越懂你
- **自我进化**：犯错后主动记录，下次避免
- **24小时在线**：心跳机制+定时任务
- **技能生态**：一句话安装各种skill
- **多端连接**：飞书、钉钉、QQ等通讯软件

### 安装配置

#### 环境要求
- Node.js 18+
- Git
- 推荐Mac系统，Windows也可

#### 配置步骤
1. 安装依赖：环境、API配置、通讯软件连接
2. 选择模型：至少需要顶尖模型+国产包月
3. 连接通讯软件：飞书、钉钉等
4. 安装基础skill：apple-notes、puppet、clock-up等

### 省钱技巧

#### 上下文瘦身
- **全身瘦身**：定期体检，清理无用工具和过时记忆
- **瘦肚子**：禁用不常用的内置工具（50多个）
- **顽固脂肪**：用CLI替代浏览器自动化
- **瘦大腿**：关闭心跳机制，用定时任务替代
- **瘦手臂**：定期compact压缩对话历史

#### 模型选择策略
- **第一档**：深度分析、复杂决策 → 高级模型（Claude、GPT-4）
- **第二档**：日常检索、写文档 → 包月套餐模型
- **第三档**：心跳检查、定时任务 → 本地模型

#### 本地模型推荐
- **硬件**：Mac Studio 128G、AMD Ryzen AI Max 395、NVIDIA DGX Spark
- **模型**：Qwen3-Coder-32B、J1-4（Google最新agent模型）

### 安全实践

#### 外部风险防护
1. **不暴露网关端口**：检查并关闭公网暴露
2. **不装来路不明skill**：只用Clothub官方市场，用skill-vetter扫描
3. **不让外部对话**：避免提示词注入攻击

#### 内部约束
- 运行安全检查
- Docker隔离执行
- 限制工具权限
- 提示词软性约束
- **最佳实践**：给Agent单独的电脑和账号

## Agent Skills 开发指南

### Skill结构

```
.claude/skills/
└── skill-name/
    ├── skill.md          # 必需：元信息+指令
    ├── references/       # 可选：参考资料
    │   ├── guide1.md
    │   └── guide2.md
    ├── scripts/          # 可选：可执行脚本
    │   └── generate.py
    └── assets/           # 可选：资源文件
        └── logo.png
```

### skill.md 结构

```markdown
---
name: skill-name
description: 技能描述，告诉AI什么时候用
---

# 指令内容
告诉AI具体怎么做...
```

### 按需加载机制（三层结构）

1. **元信息层**：始终加载，AI每次对话都会看到
2. **指令层**：AI决定使用时才加载完整skill.md
3. **资源层**：需要时才加载references/scripts/assets
   - scripts只执行不读取，不占token

### 创建Skill方法

#### 方法1：使用Skill Creator
```
帮我创建一个skill，功能是...
```

#### 方法2：手动创建
1. 在`.claude/skills/`创建文件夹
2. 编写skill.md（元信息+指令）
3. 添加references/scripts/assets（可选）
